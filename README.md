# 项目简介
超级东油是一款由东北石油大学在校生编写的长期维护的教务信息查询微信小程序。他为学生提供了方便的教务信息查询功能。
# 项目预览
整个项目包含学生账号登录绑定、上传课表、查询成绩、教务新闻浏览等功能。
